Agregat And Pattern Match

by Irfan in Design Mei 25, 2019 1020

Aggregate Functions (COUNT, SUM, AVG, MIN, MAX), ialah sebuah teknik untuk menghitung menjumlah, mengambil nilai terkecil dan terbesar, dan menghitung rata rata. Sedangkan Pattern Matching yaitu Fungsi string digunakan untuk menampilkan data yang di dasarkan pada pencarian dengan karakter. Pada pencarian data digunakan syntax LIKE, pada dasarnya syntax LIKE hampir sama dengan syntax (=) .

Tutorial Pemakaian Agregat and Pattern Match

Seperti biasa, pastikan xampp aktif dan mysql aktif, kita akan menggunakan command line di tutorial kali ini.

Adapun yang kita bahas kali ini ialah :

AGGREGAT
  1. COUNT
  2. SUM
  3. AVG
  4. MIN
  5. MAX
PATTERN MATCH
  1. Dengan huruf di awal kalimat
  2. Dengan huruf di akhir kalimat
  3. Dengan huruf di kalimat kedua
  4. Dengan huruf di posisi manapun
  5. Dengan huruf i atau o
  6. Dengan kalimat yang panjangnya 3 karakter
Didalam blog DDl Dan DML kamu telah belajar cara create database beserta tabel tabel nya, sebelum masuk ke Aggregat and Pattern Match query Buatlah Sebuah database dan tabel dengan 5 data didalam tabel nya.
COUNT

Perintah Count Di Aggregate , yaitu Menghitung Jumlah row field data dalam table yang dicari, misal nya menghitung Data dengan nama berikut ini

Perintah Count:
select COUNT(nama_kolom) from nama_tabel


SUM

Perintah SUM Di Aggregate , yaitu Menjumlahkan atau mentotalkan nilai row field data dalam table yang dicari, misal nya Menjumlahkan Data dengan Npm field berikut ini

Perintah SUM:
select SUM(nama_kolom) from nama_tabel


AVG

Perintah AVG Di Aggregate , yaitu menghitung rata rata nilai field data dalam table yang dicari, misal nya menghitung nilai rata rata data Npm field berikut ini

Perintah AVG:
select AVG(nama_kolom) from nama_tabel


MIN

Perintah MIN Di Aggregate , yaitu menampilkan nilai yang paling kecil dalam table yang dicari, misal nya nilai data Npm field berikut ini

Perintah MIN:
select MIN(nama_kolom) from nama_tabel


MAX

Perintah MAX Di Aggregate , yaitu menampilkan nilai yang paling besar dalam table yang dicari, misal nya nilai data Npm field berikut ini

Perintah MAX:
select MAX(nama_kolom) from nama_tabel


Dengan huruf di awal kalimat

Perintah ini Di Pattern , yaitu menampilkan data yang memiliki huruf yang dicari diawal kalimat, misal mencari huruf i berikut ini

Perintah:
select * (nama_kolom) from nama_tabel where like 'i%'


Dengan huruf di akhir kalimat

Perintah ini Di Pattern , yaitu menampilkan data yang memiliki huruf yang dicari diakhir kalimat, misal mencari huruf i berikut ini

Perintah:
select * (nama_kolom) from nama_tabel where like '%i'


Dengan huruf di kalimat kedua

Perintah ini Di Pattern , yaitu menampilkan data yang memiliki huruf yang dicari di kalimat kedua, misal mencari huruf a berikut ini

Perintah:
select * (nama_kolom) from nama_tabel where like '_a%'


Dengan huruf di posisi manapun

Perintah ini Di Pattern , yaitu menampilkan data yang memiliki huruf yang dicari, misal mencari huruf i berikut ini

Perintah:
select * (nama_kolom) from nama_tabel where like '%i%'


Dengan huruf i atau o

Perintah ini Di Pattern , yaitu menampilkan data yang memiliki huruf yang dicari, misal mencari huruf i atau huruf o berikut ini

Perintah:
select * (nama_kolom) from nama_tabel where like 'i%o'


Kesimpulan
