Group And Sorting

by Irfan in Design Mei 26, 2019 1020

Group and Sort adalah teknik query sql php yang mana kita mengelompokan data atau pun mengurutkan data sesuai syntax yang kita buat atau variabel yang kita tentukan.

Tutorial Pemakaian Agregat and Pattern Match

Seperti biasa, pastikan xampp aktif dan mysql aktif, kita akan menggunakan command line di tutorial kali ini.

Adapun yang kita bahas kali ini ialah :

GROUP
  1. Grouping
  2. Group dengan Having
  3. Order By
Didalam blog sebelumnya, kamu telah belajar cara create database beserta tabel tabel nya lalu mencari dengan ketentuan spesifik atau agregate pattern matching, sebelum masuk ke Group and Sort query Buatlah Sebuah database dan tabel dengan 5 data didalam tabel nya. (Jika lupa query create database dan create table bisa dibaca lagi DML dan DDL)

Grouping

Perintah Grouping Di SQL , yaitu mengelompokan kolom dengan data yang dicari, misal nya mengelompokkan Data dengan jurusan berikut ini

Perintah Group:
select nama_kolom, perintah_aggregat(nama_kolom) from nama_tabel group by nama_kolom


Group dengan Having

Perintah ini Di SQL , mengelompokan kolom dengan data yang dicari berdasarkan isi data kolom yang kita tentuin, contohnya berikut ini

Perintah Having:
select nama_kolom, perintah_aggregate(nama_kolom) from nama_tabel group by nama_kolom having perintah_aggregate(nama_field)="Isi_data_nya"


Order By

Perintah Order By Di SQL , yaitu mengurutkan data kolom yang dicari order by ada dua ketentuan yang bisa kita pakai yaitu desc dan asc, artinya kita bisa mengurutkan berdasarkan nilai terbesar dan terkecil, contohnya mengurutkan npm dengan nilai terbesar.

Perintah Order By:
select nama_kolom from nama_tabel order by nama_kolom desc


Kesimpulan

Selamat, kamu berhasil melakukan perintah Group dan Sort mysql. dengan menggunakan perintah perintah diatas kita dapat dengan mudah Mengelompokkan dan mengurutkan Pencarian di dalam Table mysql secara tepat dan tertentu.
